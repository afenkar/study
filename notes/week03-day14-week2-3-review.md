## 口述清单（15 分钟，不看笔记）

1. **bind**：和 call 差在哪？

call 立刻执行；bind 返回绑好 this 的新函数，以后再调用才执行

2. **new 四步**：默写出来

建新对象 → 连原型 prototype → 对象作为 this 调用构造函数 → 返回值是对象则用返回值，否则用第一步的新对象

3. **浅拷贝改 nest**：为什么原对象也变？

浅拷贝只有第一层是新的，嵌套对象仍共享同一引用

4. **搜索框 / 滚动**：分别用防抖还是节流？为什么？

搜索框：防抖，等停手只执行最后一次，少打接口
滚动：节流，按固定间隔执行，滚动过程中也要响应
