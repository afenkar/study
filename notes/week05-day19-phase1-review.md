## 口述 8 句（闭卷，各 1 句）

1. **闭包**：函数 + 能访问的外层词法变量
2. **this**：普通函数看调用者；箭头继承外层 this
3. **原型链**：自身 → `__proto__` → … → null
4. **new 四步**：建对象 → 连原型 → 当 this 调 → 决定返回谁
5. **深 / 浅拷贝**：浅只一层；深递归新建
6. **防抖 / 节流**：等停手 vs 按频率
7. **all / race**：全到齐 vs 谁先到
8. **事件循环**：同步 → 微任务 → 宏任务
